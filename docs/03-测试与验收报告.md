# 测试与验收报告（简版）

## 1. 测试环境
- OS：Windows 10
- Node.js：LTS
- 启动命令：`node start.js`
- 前端地址：`http://localhost:5173`
- 后端地址：`http://localhost:3000`

## 2. 功能测试清单

| 编号 | 测试项 | 预期结果 | 结果 |
|---|---|---|---|
| T01 | 注册/登录 | 可成功登录并获取 token | 通过 |
| T02 | 世界地图解锁 | 未解锁关卡不可越级（普通用户） | 通过 |
| T03 | 关卡校验 | 提交正确代码后可通关 | 通过 |
| T04 | 奖励发放 | 通关后增加 EXP，触发徽章逻辑 | 通过 |
| T05 | 排行榜 | 显示排名，普通用户不可见 admin | 通过 |
| T06 | AI 提示 | 提交错误代码可返回提示/清单 | 通过 |
| T07 | 协作房间 | 两人同房开始，状态实时同步 | 通过 |
| T08 | 协作奖励 | 每关 +20 EXP，5关“协作大师” | 通过 |
| T09 | 管理员后台 | 可查看用户并管理题库 | 通过 |
| T10 | 管理员学习特权 | 可查看参考答案，一键填入答案 | 通过 |

## 3. 兼容与边界验证
- 端口被占用：`start.js` 自动清理并重启服务。
- 多标签账号串号：登录态迁移为 `sessionStorage`。
- 协作撞房：支持随机房间号与复制分享。

## 4. 验收结论
- 项目在功能层面已满足“游戏化学习 + AI 辅导 + 协作挑战”的核心目标，可用于课程展示与毕业答辩演示。

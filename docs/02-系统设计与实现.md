# 系统设计与实现说明

## 1. 技术栈
- 前端：Vue 3 + Vite + Tailwind CSS
- 后端：Node.js + Express
- 数据库：SQLite + Sequelize
- 实时通信：Socket.IO
- AI：OpenAI 兼容接口（可接 OpenAI / 智谱 / DeepSeek）

## 2. 系统模块
- 用户认证模块：注册/登录/JWT/角色权限
- 学习模块：关卡加载、代码编辑、校验、进度保存
- AI 辅导模块：解题提示、学习路径、补强题
- 游戏化模块：EXP、等级、徽章、排行榜、烟花反馈
- 协作模块：双人房间、A/B分工、实时状态同步、奖励发放
- 管理模块：用户管理、题库管理、管理员参考答案与快捷操作

## 3. 关键流程

### 3.1 单人学习流程
1. 用户登录，加载地图与当前关卡。
2. 用户编写代码并提交校验。
3. 校验通过后更新：`solvedLevelKeys`、`EXP/Level`、徽章。
4. 弹出奖励反馈，再由用户确认是否进入下一关。

### 3.2 协作流程
1. 选择协作关卡（5关）并创建随机房间或输入房间号加入。
2. 两名用户进入同一房间后，分别分配 A/B 部分。
3. 双方提交并都通过，判定本关通关。
4. 每人发放 +20 EXP；5关全部完成发“协作大师”徽章。

## 4. 数据模型（核心）
- `users`：用户账号与角色
- `user_profiles`：学习进度、经验、等级、薄弱点、协作通关记录
- `levels`：关卡定义、初始代码、校验规则、任务说明
- `user_level_submissions`：用户每关提交代码存档
- `badges` / `user_badges`：徽章定义与授予记录

## 5. 可扩展点
- 协作历史统计（按用户/时间/队友聚合）
- 题目难度分层与自适应推荐
- 可视化学习分析看板（完成率、卡关率、复练率）
